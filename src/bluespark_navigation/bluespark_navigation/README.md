# Navigation Node

## Node used for autonomous navigation of AUV

## Running 
```bash

ros2 run navigation_node ```

## Dependencies (All should be running):
* Vision Node
* Depth Hold Node
* rc_override_node
* vehicle_manager_node

## Subscribed Topics:
* /detected_objects

## Communicating with services:
* All 6 DoFs from rcOverride node
